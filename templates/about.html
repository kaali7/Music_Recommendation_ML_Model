Music Recommendation ML Model

This project is based on a subset of The Million Song Dataset, integrating data from multiple sources such as Last.fm, Spotify, and The Echo Nest. The dataset includes information on over 50,000 songs, user listening history with nearly 10 million records, and 1,500 MP3 examples spanning 15 genres . The dataset features attributes such as track ID, song name, artist, genre, release year, duration, and several audio characteristics like danceability, energy, loudness, speechiness, and tempo .

This web project was completed in five days, during which I focused on learning web development from scratch . While the model development was relatively easy, the frontend development was quite challenging. The project is built using Flask for the backend, while the frontend utilizes HTML, CSS, Bootstrap, and JavaScript. The recommendation system is powered by a machine learning model using the Nearest Neighbors algorithm from scikit-learn, and it employs LabelEncoder for processing categorical data . The application suggests songs based on their audio features, enhancing user experience with music recommendations .
